*** 
- Converts .py source code into bytecode (.pyc files).
- This bytecode is then executed by the Python Virtual Machine (PVM).
- The compilation step is automatic and hidden from the programmer.
(#) is used for single line comment
(''' ''') for multiple line comment
Note: this comments are ignored by compiler and interpreter
